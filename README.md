# 数独求败

稳定母版：**v3.0.0**

当前开发版本：**v3.0.14**

题库同时按挖空数量、实际技巧路径和专业难度评级分级；不允许用无限推演制造难度。王者题开局不得存在显性唯一或隐性唯一，求败题必须在基础技巧后形成高级技巧阻断。本版求败模板采用公开题库中 Sukaku Explainer 7.2 级逻辑题，必须继续使用翼、链等组合技巧才能完成。

候选删除技巧采用两阶段识别：三数组形成并产生有效候选删除后，玩家手动删除对应标记或填入由此释放的唯一数字，都会记录三数组、触发解锁提示并发放技巧奖励。

一个无需安装依赖的中文数独网页游戏。

## 运行

直接双击 `index.html`，或在本目录启动任意静态文件服务器。

功能包括 9×9 数独、初级/中级/高级/王者/求败五档题目、逐级试玩解锁、计时、积分档案、错误提示、每局 3 次提示、完成检测、键盘操作、明暗主题以及手机/电脑响应式布局。

## 发布

确认本地版本可发布后，在 PowerShell 进入项目目录，运行：

```powershell
powershell -ExecutionPolicy Bypass -File .\publish.ps1
```

按提示粘贴 GitHub token。脚本会自动提交并覆盖 GitHub Pages 对应仓库，不会保存 token。
