## v3.0.0 稳定范围

v3.0.0 锁定以下已确认内容：

- 中文 / 英文双语界面。
- 9×9 数独主玩法。
- 初级 / 中级 / 高级 / 王者 / 求败五级体系。
- 积分、升级、降级、试玩挑战规则。
- 错误 3 次结束本轮，可扣积分继续。
- 标记、擦除、一键标记、快速填充等功能规则。
- 技巧提示、技巧勋章、技巧奖励、AI 简单互动。
- 完成页、失败页、奖励页、玩家档案等核心页面。
- 手机、电脑、右侧窄窗的一屏自适应布局。
- 当前题库设计方向：不是为了难住玩家，而是通过题型训练帮助玩家掌握技巧。

## 后续开发原则

1. **v3.0.0 作为稳定母版**
   - 后续小程序、App、线上网页版发布，都应先对齐 v3.0.0 的玩法、积分、等级、标记、提示、技巧识别和双语界面。
   - 新功能只能在此基础上新增，不能无意覆盖已确认规则。

2. **已确认规则不可随意改动**
   - 如果要调整等级、积分、题库、技巧提示、快速填充、标记逻辑等核心玩法，必须先明确写成新版本规则。
   - 修改前应说明“改动什么、为什么改、会影响哪些旧规则”。

3. **发布前必须做同步检查**
   - 本地版、网页版、小程序版、App 版必须逐项检查核心功能是否一致。
   - 不允许出现“本地能用，线上没有同步”的情况。

## 下一阶段学习方向

### 小程序发布

建议优先学习微信小程序发布流程：

1. 注册微信小程序账号。
2. 安装微信开发者工具。
3. 把当前网页游戏改造成小程序页面结构。
4. 适配小程序的数据存储、按钮、音效、分享。
5. 上传体验版。
6. 提交审核。
7. 发布正式版。

### App 发布

建议先走“网页封装 App”的路线，速度更快：

1. 保持当前网页游戏作为核心。
2. 用 Capacitor / Cordova / Tauri 等方式封装。
3. 先打包安卓 APK 测试。
4. 再考虑 Android 应用市场和 iOS App Store。
